## ⚡ Overview

IGCSEPulse is a modern AI platform that combines:
- 💬 Real-time AI chat  
- 📄 Document ingestion & processing  
- 🔍 Semantic vector search  
- 🧠 Multi-model AI routing  

Built with a **modular, API-first architecture** for rapid iteration and scalability.

---

## 🧠 System Architecture

Client (Next.js UI)  
↓  
API Routes  
↓  
AI Providers (Mistral / Groq / Others)  
↓  
Pinecone Vector DB  
↓  
Response → UI
